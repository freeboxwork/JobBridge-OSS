# Model card: lightgbm_jobseeker_preference_v1

## 요약

장애인 구직자가 등록한 희망직종 대분류의 분포를 지역·연령·장애유형·중증도에서 추정하는 LightGBM 다중분류 모델입니다. 추천 순위를 보조하는 prior이며 취업 성과·능력·적성을 예측하지 않습니다.

## 파일과 라이선스

- 가중치: `jobbridge_preference_model.joblib`
- 프레임워크: LightGBM 4.6.0, scikit-learn 1.7.2, joblib 1.5.3
- 참가자 작성 코드·가중치: Apache-2.0
- 학습 원천: 한국장애인고용공단 「장애인 구직자 현황」, 공공누리 제1유형
- 원천 URL: https://www.data.go.kr/data/15014774/fileData.do

## 입력과 출력

입력은 `sido`, `sigungu`, `age`, `age_group`, `disability_type`, `severity`입니다. 출력은 28개 직무 대분류에 대한 확률입니다. 서비스는 사용자의 명시적 희망직무와 임금을 별도 후처리로 우선 반영합니다.

## 학습·평가

- 전체 41,719행, stratified 80/20 holdout, seed 42
- 학습 33,375행, 평가 8,344행
- 최적 반복 310
- Top-1 0.2604, Top-3 0.5585, Top-5 0.7270
- Balanced accuracy 0.2605, macro F1 0.1833, weighted F1 0.2860
- Multiclass log loss 2.4857

라벨은 희망직종 원문을 규칙·직종코드로 매핑한 약한 라벨입니다. 정확도는 개인 수준의 의사결정에 충분하지 않으며 성능 수치를 과장해서는 안 됩니다.

## 제한과 위험

- 과거 등록 선호의 직업 성별화·지역 격차·정보 접근 격차를 재생산할 수 있습니다.
- 장애유형과 인구통계는 개인의 능력을 설명하지 않습니다.
- 희소 직무의 성능이 낮고 대분류 간 표본 불균형이 큽니다.
- 실제 채용 결과·장기 고용 유지·만족도를 검증하지 않았습니다.
- 안전·접근성 판단을 자동화하거나 고용주 선별에 사용하면 안 됩니다.

## 허용 사용

- 구직자가 직접 사용하는 탐색 화면의 보조 순위
- 여러 직무군을 제안하는 아이디어 확장
- 연구·교육용 재현과 편향 분석

## 금지·부적절 사용

- 채용 여부나 교육 기회를 자동 결정하는 고위험 의사결정
- 장애유형만으로 직업을 배제하거나 제한하는 필터
- 사용자 동의 없는 프로파일 저장·결합
- 의료·법률적 판단

## 완화책

- 사용자가 선택한 직무를 모델보다 우선
- Top-k와 근거·주의 문구를 함께 표시
- 합성 공고를 기본값으로 사용
- 원본·중간 행 데이터를 Git에 미포함
- 추천 로그 기본 비활성화
- 당사자 피드백과 직무별 공정성 평가를 후속 과제로 유지
