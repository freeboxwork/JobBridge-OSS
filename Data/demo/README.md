# Synthetic demo data

All rows in this directory are fictional and were created only to run JobBridge
without downloading third-party datasets. They do not describe real people,
employers, phone numbers, or vacancies.

The demo data is licensed under Apache-2.0 with the rest of this repository.
Production integrations must follow the source terms documented in
[`docs/DATA_SOURCES_AND_LICENSES.md`](../../docs/DATA_SOURCES_AND_LICENSES.md).
