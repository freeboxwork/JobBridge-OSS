# JobBridge 3분 시연영상 대본

목표 길이: 2분 50초~2분 58초
주의: 개인정보·비밀키·관리자 화면·실제 연락처를 녹화하지 않습니다.

## 0:00~0:20 문제와 해결

화면: JobBridge 첫 화면과 한 줄 문제정의

> 장애인 구직자는 공고가 있어도 자신의 희망조건과 작업환경 정보를 함께 비교하기 어렵습니다. JobBridge는 지역·직무·임금·접근성 단서를 한 화면에 모아 탐색 비용을 줄이는 오픈소스 서비스입니다.

## 0:20~0:48 프로필 입력

화면: 합성 시연 프로필 입력

- 지역: 경기 수원시
- 연령: 32세
- 장애유형: 청각장애
- 중증도: 경증
- 희망직무: 경영·행정·사무직
- 희망임금: 월 220~260만원

> 장애유형은 능력을 판정하는 값이 아닙니다. 사용자가 고른 희망직무와 조건이 모델보다 우선하며 입력값 저장은 기본적으로 꺼져 있습니다.

## 0:48~1:25 추천과 설명

화면: 직무군 Top-K와 추천 카드

> 공개 LightGBM 모델은 공공누리 제1유형 구직자 데이터에서 등록 희망직종 분포만 학습한 보조 신호입니다. 취업 성공률을 예측하지 않습니다. 결과에서는 모델 확률과 사용자 선호 boost, 지역·임금 일치, 주의할 작업조건을 분리해 보여 줍니다.

## 1:25~1:55 공고 비교

화면: 합성 공고 상세·지도 링크·작업환경

> 기본 공개 실행에는 실제 기업이 아닌 합성 공고 10건이 포함됩니다. 근무지와 임금뿐 아니라 양손작업, 시력, 의사소통, 서기·걷기 같은 작업환경 단서를 함께 비교할 수 있습니다.

## 1:55~2:20 도전추천과 준비 경로

화면: 역량 선택·도전추천 베타

> 현재 역량과 관심기술을 선택하면 NCS 기반 준비 경로를 제안합니다. 이 기능은 베타이며 직무를 차단하지 않고 다른 가능성을 탐색하도록 돕습니다.

## 2:20~2:43 공개 저장소 재현

화면: GitHub README → 모델 카드 → Actions

> GitHub에는 웹·API·학습 코드, 13MB 공개 가중치, 합성 데이터, 자동 테스트와 재현 명령을 공개했습니다. 제2유형 비상업 데이터와 기존 파생 모델은 심사 공개본에서 제거했습니다.

## 2:43~2:58 효과와 로드맵

화면: 효과 지표 3개

> 다음 단계는 장애 당사자와 함께 탐색시간, 추천 이해도, 조건 확인 누락률을 측정하고 집단별 공정성을 검증하는 것입니다. JobBridge를 지역 취업지원기관이 재사용할 수 있는 공공 오픈소스로 키우겠습니다.
