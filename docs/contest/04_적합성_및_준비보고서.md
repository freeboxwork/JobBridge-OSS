# JobBridge 공모전 적합성 및 제출 준비 보고서

작성일: 2026-07-14

## 결론

**적합합니다.** JobBridge는 장애인 등 사회적 약자의 생활 속 불편 개선을 명시한 사회문제해결 `생활` 분야에 직접 부합합니다. 실제 접수에서는 독립 과제유형이 아니라 `자유과제 + 사회문제해결 해당 + 생활` 조합으로 선택하는 것이 현재 UI 기준입니다.

## 적합 근거

- 문제: 장애인 구직자의 채용정보·작업환경 정보 분절과 높은 탐색 비용
- 대상: 장애인 구직자와 취업지원사
- 해결: 프로필·희망조건·접근성 단서를 결합한 비교·추천·설명
- 사회적 가치: 정보 접근성 향상, 상담 보조, 지역기관의 오픈소스 재사용
- 오픈소스성: 전체 웹·API·학습 코드, 공개 가중치, 재현 문서와 테스트 제공

공식 근거:

- https://osscontest.kr/overview
- https://osscontest.kr/tasks
- https://api.osscontest.kr/static/uploads/b3b4491a-3bbe-454e-a1d8-6ed475b01b14.pdf

## 공개 전 감사에서 발견한 위험과 조치

| 위험 | 조치 |
|---|---|
| 기존 저장소의 브라우저 프로필·실명 문서·작업 산출물 | 기존 저장소를 공개하지 않고 clean 공개본 생성 |
| 공공누리 제2유형 취업성공·구인 데이터 | 원본·가공본·파생 모델을 공개본과 심사 기본 실행에서 제외 |
| 모델 표현이 취업 성공·적성 판단으로 오해될 위험 | 제1유형 희망직종 데이터로 재학습하고 preference prior로 명시 |
| 추천 입력 저장 고지 불일치 | 공개본 로그 기본값 `false`와 개인정보 원칙 문서화 |
| 버전이 고정되지 않은 CDN | Supabase JS·Lucide 등 버전 고정 |
| README·LICENSE·SBOM·모델카드 부재 | Apache-2.0과 제출 필수 문서 작성 |
