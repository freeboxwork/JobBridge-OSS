안녕하세요. 2026 오픈소스 개발자대회에 장애인 구직자의 채용정보 접근성과 탐색을 돕는 JobBridge를 자유과제·사회문제해결(생활)로 접수하려고 합니다.

다음 세 가지를 확인 부탁드립니다.

1. 현재 접수 UI에서 `자유과제`를 선택한 뒤 `사회문제해결 프로젝트 해당`과 `생활` 분야를 함께 선택하는 방식이 맞는지요?
2. 최종 제출 ZIP에는 결과보고서 DOCX와 PDF 외에 소스코드 아카이브도 포함해야 하나요, 아니면 보고서의 public GitHub URL로 전체 소스 제출을 갈음하나요?
3. 공공누리 제2유형(출처표시·상업적 이용금지) 공공 API를 비상업 공모전 시연의 선택 기능으로 호출하는 것이 허용되나요? 해당 원본과 파생 모델은 공개 저장소·기본 심사 실행에서 제외하고, OSI 라이선스 코드와 공공누리 제1유형 기반 공개 가중치만 제출할 예정입니다.
