# 데이터 출처와 라이선스

기준일: 2026-07-14

이 문서는 JobBridge의 데이터 사용 경계를 재현 가능하게 기록합니다. 저장소의 Apache-2.0은 참가자가 작성한 코드와 명시적으로 표시된 합성 데이터·모델 가중치에만 적용되며, 외부 공공데이터의 원 라이선스를 덮어쓰지 않습니다.

2026 오픈소스 개발자대회 운영규정은 참가자가 작성한 소스에 OSI 승인 라이선스를 요구하고, 비상업 전용 라이선스의 사용을 원칙적으로 제한합니다. 따라서 위 제2유형 데이터에 의존하는 기존 성과예측 모델과 공고 랭커는 공모전 공개본에서 제거했습니다.

## 공개 모델의 데이터 흐름

1. 사용자가 공공데이터포털에서 제1유형 구직자 CSV와 직종코드 CSV를 직접 내려받습니다.
2. `Scripts/prepare_oss_training_dataset.py`가 직무명을 대분류로 매핑합니다.
3. 연번, 희망직종 원문, 희망임금, 기관분류를 학습 산출물에서 제거합니다.
4. 지역·연령대·장애유형·중증도에서 등록 희망직종 대분류의 분포를 학습합니다.
5. 원본과 중간 행 단위 데이터는 Git에 커밋하지 않습니다.

이 라벨은 취업 성공이나 직무 적합성 라벨이 아닙니다. 서비스는 이를 낮은 가중치의 선호 사전분포로만 사용하고, 사용자가 고른 희망 직무·임금과 접근성 조건을 우선합니다.

## 선택형 실시간 연동

`Scripts/collect_kead_live_jobs.py`와 관련 서버 코드는 비상업 조건이 있는 공공 API 연동 예제를 포함할 수 있으나, 공개본 기본 실행에서는 비활성화되어 있고 해당 데이터를 재배포하지 않습니다. 실제 운영자는 자신의 이용 목적이 원 라이선스에 맞는지 확인하고 출처를 표시해야 합니다.

공모전 심사 범위에는 `Data/demo`와 공개 선호 모델만 포함합니다. 제2유형 실시간 연동을 심사 시연에 포함하려면 먼저 운영사무국과 데이터 제공기관의 서면 확인을 받습니다.

## 표준사업장 데이터

[장애인 표준사업장 현황](https://www.data.go.kr/data/3033670/fileData.do)은 공공누리 제1유형이지만 전화번호와 사업자번호가 포함될 수 있어 원본을 저장소에 넣지 않았습니다. 선택적으로 다운로드해 서버에서만 사용할 수 있습니다.

## 업데이트 절차

- 데이터 파일이 갱신되면 공식 페이지의 이용조건과 기준일을 다시 확인합니다.
- `dataset_metadata.json`에 원본 SHA-256, 행 수, 분류 수를 기록합니다.
- 새로운 외부 데이터는 이 표와 `THIRD_PARTY_NOTICES.md`에 추가한 뒤 사용합니다.
- 라이선스가 불명확하면 저장소와 모델에서 제외하고 권리자에게 서면 문의합니다.
